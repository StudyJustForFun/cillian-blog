Markdown 写作：程序员的文档神器，普通人也能用

2026-03-24 技术 工具

如果你还在用 Word 排版文档，那你可能错过了一个更好的工具。

Markdown ——一种用纯文本写格式化文档的语言。程序员用它写文档、写博客、写小说；普通人用它写简历、写笔记、写公众号。

什么是 Markdown？

Markdown 是一种「轻量级标记语言」。你可以把它理解为：用普通文本 + 一些特殊符号，就能写出带格式的文档。

比如：

# 标题
## 二级标题
**加粗文字**
*斜体文字*
- 列表项
1. 有序列表
[链接文字](https://example.com)
![图片](image.jpg)
> 引用文字
`代码`

为什么程序员爱用它？

1. 纯文本，永久保存

Word 文件用新版 Word 打开可能变形。用 Markdown 写的文档，50 年后依然可以用任何文本编辑器打开。

2. 专注内容，不用管格式

Word 里调格式的时间比写作还长。Markdown 让你专注写作，格式只是符号。

3. 版本控制友好

用 Git 管理 Markdown 文档，可以看到每次修改了什么。多人协作也不怕冲突。

4. 一键转多种格式

一篇 Markdown，可以一键转成 HTML、PDF、Word、PPT。用对了工具，30 秒搞定排版。

普通人用 Markdown 能做什么？

  • 写公众号文章 —— 用 Markdown 写，粘贴到公众号编辑器
  • 写简历 —— 用 Markdown 写简历，简洁大方，ATS 友好
  • 写笔记 —— Obsidian、Notion、Typora 都支持 Markdown
  • 写小红书 —— 小红书原生支持 Markdown 格式
  • 写技术文档 —— GitHub、GitLab 都用 Markdown 写文档

快速入门：5 分钟学会基础语法

标题

# 一级标题
## 二级标题
### 三级标题

文字样式

**加粗** 或 __加粗__
*斜体* 或 _斜体_
~~删除线~~

列表

- 无序列表项
- 另一个列表项

1. 有序列表
2. 第二个

链接和图片

[链接文字](https://example.com)
![图片说明](image.jpg)

代码

`行内代码`

```
代码块
多行代码
```

引用

> 这是一段引用文字

推荐工具

写作工具

  • Typora —— 所见即所得，免费好用
  • Obsidian —— 笔记软件，支持双链
  • Notion —— 全能工具，支持 Markdown
  • VS Code —— 程序员首选，有 Markdown 预览

在线工具

  • mdnice —— Markdown 转公众号格式，支持多种主题
  • StackEdit —— 在线 Markdown 编辑器
  • Dillinger —— 简洁好看的在线编辑器

我的写作流程

这个博客的所有文章都是用 Markdown 写的：

  1. 用 Typora 或 VS Code 写 Markdown
  2. 生成 HTML 文件
  3. 部署到博客
  4. 发布

整个过程不需要关心格式，只需要专注写作。

总结

  • Markdown = 纯文本 + 格式符号
  • 程序员写文档的首选，普通人也好上手
  • 永久保存、版本控制、一键转格式
  • 推荐工具：Typora、Obsidian、mdnice

花 30 分钟学一下 Markdown，你会回来感谢我的。
