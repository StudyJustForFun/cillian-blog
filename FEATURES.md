# Cillian.xin 功能开发路线图

> 本文档记录网站所有功能更新的规划、进度和实现细节

## 设计原则

1. **一致性**：所有功能使用统一的深色主题 + 紫色强调色
2. **轻量级**：纯前端实现，无后端依赖，CDN 友好
3. **渐进增强**：基础功能优先，高级功能可选
4. **测试先行**：本地测试通过后再推送上线
